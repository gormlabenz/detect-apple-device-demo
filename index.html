About This Demo

This demo showcases the detect-apple-device package, which approximates Apple device identification based on screen metrics. The library uses:

  • Screen Width & Height: Logical pixel dimensions
  • Device Pixel Ratio: Scale factor (1x, 2x, 3x)
  • Privacy-First: No fingerprinting or tracking
